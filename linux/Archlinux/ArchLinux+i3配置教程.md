# exec命令

```
exec 执行命令
   --no-startup-id 有些脚本或者程序不支持启动通知，不加命令，鼠标会长时间空转，60秒左右
   exec_always 每次重启i3，使用该命令启动的程序都会重新执行一次，不然只在开机时执行一次。
```

# 触控板配置

```
我电脑上使用的触控版驱动是xf86-input-libinput
要修改单击=鼠标左键 双击=鼠标右键 参考wiki

https://wiki.archlinux.org/index.php/Libinput_(%E7%AE%80%E4%BD%93%E4%B8%AD%E6%96%87)

$ # ln -s /usr/share/X11/xorg.conf.d/40-libinput.conf /etc/X11/xorg.conf.d/40-libinput.conf 
$ sudo vim /etc/X11/xorg.conf.d/40-libinput.conf
#在下面的节点中添加片段： 

Section "InputClass" 
    Identifier "touchpad" 
    Driver "libinput" 
    MatchIsTouchpad "on" 
    Option "Tapping" "on" //开启触控板点击 
    Option "TappingButtonMap" "lrm" //对应两种模式，（一指：鼠标左键 双指：鼠标右键 三指：鼠标中键，对应参数lrm）（单指：鼠标左键 双指：鼠标中键 三指：鼠标右键，对应参数lmr）我要的是，双击触控板=鼠标右键，选lrm  set 1/2/3-finger taps to left/right/middle set TappingButtonMap to lrm, for left/middle/right set it to lmr. 
EndSection
```
